怎么判断 MCP 工具能不能用？

MCP 工具是给前台 AI / 数字员工调用的“能力插件”。内置工具调用本地 Python 函数； API 工具会把参数组装成 HTTP 请求；Crawl4ai 工具用于更完整的网页正文采集。

1. 已启用
工具卡片状态为“已启用”，否则不会进入 MCP Server。
2. 已加载
修改配置后点击“热重载”，看到“已加载”才会被对话链路使用。
3. 测试成功
点“测试”，按 Input Schema 填 JSON，例如 {"keyword":"AI","limit":5}。
4. 授权员工
到“数字员工 → 编辑 → MCP 工具权限”勾选工具，未勾选则员工无权调用。
💡 只想配置大模型 API 时进入 模型 API 配置；只有要新增/启停/测试可调用工具时才需要改这里。
