Python 转换脚本，函数签名: def transform(data_sources: list[dict]) -> str
📜 脚本编写指南

脚本用于 MCP 工具的 transform_script 字段，负责将接口返回的原始数据转换为最终输出。

输入数据

脚本接收一个参数 data_sources: list[dict]，每个元素是 {"success": bool, "data": any}：

data_sources = [
    {
        "success": True,
        "data": <接口返回的原始数据>   # JSON对象、数组、字符串等
    },
    ...
]

通常取第一个数据源：data_sources[0]["data"]

输出格式

脚本必须返回一个字符串。可以是：

  • JSON 字符串（推荐）：return json.dumps({"name": "xxx", "artist": "xxx"}, ensure_ascii=False) — 前端可渲染为卡片
  • 纯文本：return f"🎵 {name} — {artist}" — 直接作为对话文本

可用内置对象

以下对象已预注入，无需 import，直接使用：

json（JSON处理）· random（随机数）· re（正则）· str, int, float, bool, list, dict, tuple, set · len, range, enumerate, zip, sorted, min, max, sum, abs, round · Exception, ValueError, TypeError, KeyError

安全限制

  • 🚫 禁止 import / from ... import
  • 🚫 禁止 exec / eval / open / compile
  • 🚫 禁止 while 循环，for 循环可用
  • ✅ 纯数据转换（JSON格式化、字符串拼接、条件判断、列表处理）

示例：天气格式化

def transform(data_sources):
    # 透传wttr.in原始JSON，前端自动渲染天气卡片
    if not data_sources or not data_sources[0].get('success'):
        return json.dumps({"error": "数据获取失败"}, ensure_ascii=False)
    return json.dumps(data_sources[0].get('data', {}), ensure_ascii=False)

示例：音乐推荐

def transform(data_sources):
    songs = data_sources[0].get('data', [])
    if isinstance(songs, list) and len(songs) > 0:
        song = random.choice(songs)
        return json.dumps({
            "name": song.get("name", ""),
            "artist": song.get("artist", ""),
            "cover": song.get("pic", ""),
            "url": song.get("url", ""),
            "source": "网易云音乐热歌榜"
        }, ensure_ascii=False)
    return json.dumps({"error": "未找到歌曲"}, ensure_ascii=False)
